# plate-calculator

パワーリフティング／ウエイトトレーニング向けのプレート計算アプリ。
目標重量を入力すると、バーベルの左右に付けるプレートの組み合わせを計算します。

> 🚧 開発中（2026年8月末公開予定）。このREADMEは仮置きで、正式版はリリース時に整備します。

## 技術スタック

- Vite / React / TypeScript
- Tailwind CSS
- Vitest（テスト）
- ESLint + Prettier + husky（品質チェック）
- GitHub Actions（CI）

## 開発

```bash
npm install
npm run dev          # 開発サーバー起動
npm test             # テスト実行
npm run lint         # ESLint
npm run format:check # フォーマットチェック
```
